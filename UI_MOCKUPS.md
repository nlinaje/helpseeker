**How to Play:**
1. Tap a card to flip it over
2. **Say the name of the picture out loud** (e.g., "Apple")
3. Tap a second card and say its name
4. If they match, they stay face-up!
5. If not, try to remember where they are

**Speech Recognition:**
- ✅ Correct: "Apple" → Continue to next card
- ❌ Not recognized: "Ahhh..." → Try again button
- 💡 Help: "Say it for me" → Character says it first
- ⏭️ Skip speech: Available for non-verbal children

**Therapeutic Benefits:**
- Memory and concentration
- Vocabulary building
- Speech practice
- Sound-object association
- Turn-taking patience

- Fruits & Vegetables (🍎 🍌 🥕 🍇)

```

**Incorrect Answer Flow:**
```
Child sees: 2 + 3
Child says: "Six"

- Math skills practice

- Number recognition
- Quick thinking
- Confidence building
